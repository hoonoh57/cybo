# Design Decision: CYBO Architecture Initialization

## Status: APPROVED
## Date: 2026-03-30
## Deciders: AI Engine + User

## Context
Initial design of the CYBO AI video editor.

## Decision
- Use functional blocks: Atoms, Molecules, Organisms, and Templates.
- Use Puck for visual editing.
- Use CSS Modules for styling with Design Tokens.
- Storybook for visual testing of every component.
- Claude Code for automated project context and workflows.

## Rationale
- Modularity is essential for scaling the project.
- Visual editing (Puck) provides a low-barrier, yet powerful editor.
- CSS Modules offer modern, modular styling with strong variable support.
- Automated dev workflows ensure project consistency.

## Consequences
- Every new feature should be categorized into a block.
- Development requires registration in the Puck editor for full benefit.
