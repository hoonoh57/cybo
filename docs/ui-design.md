# CYBO UI Design Documentation

## Design Goals
- **Simplicity:** Minimize UI clutter while providing powerful features.
- **Aesthetics:** High-quality visual style with dark mode support.
- **Efficiency:** Fast workflows and responsiveness.
- **Modularity:** Everything is a block.

## Layout Overview
- **TopBar:** Quick actions and file management.
- **LeftPanel:** Library and assets.
- **PreviewArea:** Video output and composition.
- **Timeline:** Frame-accurate video sequences.
- **QuickToolbar:** Rapid tool access.
- **StatusBar:** System status and progress.

## Interactive States
- **Hover:** Subtle change in background or border.
- **Active:** Distinct highlight for selected elements.
- **Disabled:** De-emphasized state for unavailable actions.

## Design Patterns
- **Cards:** Used for media clips and project items.
- **Modals:** For complex interactions (Export, AI Magic).
- **Tabs:** To organize content in panels.

This document is updated based on new features and design decisions.
