# Command: /benchmark [product]

When this command is run:

1. Identify the product (e.g., Edimakor, CapCut, Premiere).
2. Create/update a file in `docs/benchmarks/[product].md`.
3. Analyze key features:
   - UI Layout (Panels, Timeline, Toolbar)
   - Color Palette and Visual Style
   - Interaction Patterns (Drag-and-drop, Hotkeys)
   - Unique Features (AI tools, Effects)
4. Correlate with CYBO goals and features.
5. Create a comparison table if possible.
